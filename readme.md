# PixelMarket 🎨

Application web permettant aux photographes et artistes numériques de vendre, exposer et promouvoir leurs œuvres en ligne (photos, vidéos, musique, affiche, images).

## Stack du projet

- **Backend** — Python / Flask
- **Base de données** — SQLite / SQLAlchemy
- **Auth** — Flask-Login + bcrypt
- **Emails** — Flask-Mail

## Fonctionnalités

- Parcourir et filtrer les œuvres (photos, vidéos, musique, affiches)
- Créer un compte et acheter une œuvre
- Espace artiste pour gérer ses œuvres
- Panel administrateur
- Newsletter

## Installation

```bash
git clone https://github.com/Mashiro3131/PixelMarket.git
cd PixelMarket
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python run.py
```
